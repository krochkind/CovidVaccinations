The Original_Data_Sets folder contains the data sets that were download from the internet
The Output_CSV folder holds the cleaned, transformed data that is used in the Shiny app

1. Introduction.r - Gets the RepublicanPercent (37%) and DemocratPercent (63%) vaccination rates that are hardcoded in the waffle plot in the Shiny app
2. Map.r - Outputs cleaned/transformed versions of CSV files used to create the maps.  This is also the source of the hardcoded US Vaccination Percent (67.1%)

Note: The r code to generate the State Comparisons are in the Shiny app (the source of the hardcoded Red_State_Avg (59.1%) and Blue_State_Avg (71.8%))
